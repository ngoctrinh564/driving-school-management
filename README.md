## 📌 About the Project

**Driving School Management System** is a comprehensive solution built to streamline the management of driving education. From the moment a student registers to the point they receive their license, every step is digitalized.

- **For Students:** Manage dossiers, study theory with flashcards, and practice simulation exams.
- **For Admins:** Monitor registrations, manage course schedules, and track financial reports.
- **Architecture:** Layered design (Controllers → Services → Data) for maximum scalability.

---

## ✨ Key Features

- 🔐 **Secure Auth:** Identity management with OTP verification.
- 👤 **Dossier Management:** Digital student profiles and automated document handling.
- 📖 **Smart Learning:** Interactive question banks, theory practice, and mock exams.
- 🎮 **Simulation Tests:** Realistic exam workflows with detailed history tracking.
- 💳 **Payment Hub:** Integrated with **VNPay, MoMo, and PayPal**.
- 🧾 **Auto-Invoices:** Instant PDF invoice generation using QuestPDF.
- 🤖 **AI Assistant:** Integrated chatbot for 24/7 student support.
